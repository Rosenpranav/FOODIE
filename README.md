# Food Delivery System

A simple Python-based food delivery system with MVC architecture.

## Features

- Register customers and restaurants
- Add food items
- Place orders
- View all users, foods, and orders

## Structure

- `main.py`: Main application entry point
- `models/`: Data models (User, Food, Order, etc.)
- `repositories/`: Data access layer
- `controllers/`: Business logic
- `views/`: User interface (currently minimal)

## Requirements

- Python 3.x

## Usage

Run `python main.py` to start the interactive menu.
